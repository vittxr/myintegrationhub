## Summary

To use habitica API, you need to get the API KEY, USER ID and CLIENT

## API Key

1. Access https://habitica.com/user/settings/siteData
2. Click in "learn more" btn and your API key should be available in "API Token" input.

## User ID

1. Access https://habitica.com/user/settings/siteData and your User ID should be visible in the top of the page

## Client

Client is just an ID of your application, so habitica can identify you. You can use just use `USER_ID + APP_NAME`
